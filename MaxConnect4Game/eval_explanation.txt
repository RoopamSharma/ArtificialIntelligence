The evaluation function scores are calculated in the following ways:
1. If computer has 4 in a row horizontally, vertically, diagonally left or diagonally right then +100 points.
2. If human has 4 in a row horizontally, vertically, diagonally left or diagonally right then -100 points.
3. If computer has 3 in a row horizontally, vertically, diagonally left or diagonally right and there is possibility of making 4 in a row then +50 points.
4. If human has 3 in a row horizontally, vertically, diagonally left or diagonally right and there is possibility of making 4 in a row then -50 points.
3. If computer has 2 in a row horizontally, vertically, diagonally left or diagonally right and there is possibility of making 4 in a row then +30 points.
4. If human has 2 in a row horizontally, vertically, diagonally left or diagonally right and there is possibility of making 4 in a row then -30 points.

Using the above eval function the computer is able to consistently beat human player playing randomly.
